# 🛒 Amazora — Modern E-Commerce Web App

Amazora is a fully responsive, feature-rich **Amazon-inspired e-commerce web application** built using **HTML, Tailwind CSS, and JavaScript**. It provides a complete shopping experience including product browsing, cart management, checkout, and an admin panel — all in a single-page application. Md Methun Hossain  

---

## 🚀 Features

### 🏠 User Features

* 🔍 Smart product search with suggestions
* 🛍️ Category-based filtering (Electronics, Fashion, Home, etc.)
* ⭐ Product ratings & reviews system
* ❤️ Wishlist functionality
* 🛒 Add to Cart & Buy Now
* 📦 Order management system
* 🧾 Checkout with form validation
* 🧠 AI-like recommendations (based on activity)
* 🔁 Recently viewed & Buy Again sections

### 📊 Product Listing & Filters

* Sort by:

  * Price (Low → High / High → Low)
  * Rating
  * Featured products
* Filter by:

  * Category
  * Price range
  * Customer rating

### 🔥 Advanced UI Sections

* Flash Deals with timer ⏱️
* Trending products
* Best Sellers
* Recommended products

### 👤 User Dashboard

* Orders history
* Wishlist
* Recently viewed items
* Address management
* Profile info

### 🛠️ Admin Panel

* Add / Edit / Delete products
* View product list
* View recent orders
* Dashboard statistics

---

## 🧱 Tech Stack

* **Frontend:** HTML5, Tailwind CSS
* **Icons:** Font Awesome
* **Fonts:** Google Fonts (Inter)
* **Logic:** Vanilla JavaScript
* **Storage:** LocalStorage (for state management)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/amazora.git
```

2. Open the project:

```bash
cd amazora
```

3. Run the project:

* Simply open `index.html` in your browser

## 💾 Data Storage

* Uses **LocalStorage** to store:

  * Cart items
  * Wishlist
  * Orders
  * User profile
  * Product data

⚠️ Note: Data will reset if browser storage is cleared.
